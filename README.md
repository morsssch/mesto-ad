Описание

Проект представляет собой интерактивную веб-страницу — сервис для обмена фотографиями. Пользователи могут добавлять фотографии, удалять их, ставить лайки, а также редактировать свой профиль и аватар. Проект реализован с использованием модульного JavaScript, кастомной валидации форм и сборки через Vite.
Ключевые особенности

Технологии

    - HTML5: Семантическая разметка
    - CSS3: Flexbox, Grid Layout, адаптивная верстка
    - JavaScript (ES6+): Модульная архитектура, обработка событий, DOM-манипуляции
    - Vite: Современный сборщик для разработки и production-сборки
    - Валидация форм: Кастомная реализация с поддержкой различных типов полей

Функциональность
Основные возможности

    - Просмотр карточек: Шесть предустановленных карточек при загрузке страницы
    - Добавление карточек: Создание новых карточек с изображениями
    - Редактирование профиля: Изменение имени и информации о пользователе
    - Изменение аватара: Обновление изображения профиля
    - Лайки: Взаимодействие с контентом через систему лайков
    - Просмотр изображений: Открытие полноразмерных изображений в модальном окне
